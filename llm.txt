# Homun-Lang — User Guide for Writing .hom Code

You are writing .hom code. You do NOT need to understand the compiler.
Think of .hom as "Python-like syntax that compiles to Rust."

## Quick Start

```bash
homunc main.hom -o main.rs    # compile .hom → .rs
rustc main.rs -o main          # compile .rs → binary
```

## Syntax Cheat Sheet

### Variables (`:=` immutable, `::=` mutable)
```hom
x := 10                       // immutable (let)
name := "hello"
speed := 3.14
active := true
ch := 'a'                     // char (single quotes)
nothing := none

counter ::= 0                 // mutable (let mut)
counter ::= counter + 1       // reassignment (same name in scope)
```

### String Interpolation
```hom
msg := "Hello ${name}, score: ${x * 2}"
```

### Collections (@ prefix, bracket shape = type)
```hom
list := @[1, 2, 3]                    // Vec (ordered)
dict := @{"a": 1, "b": 2}            // HashMap (key-value)
set  := @("x", "y", "z")             // HashSet (unique)
```

### Functions (lambdas, last expression = return value)
```hom
add := (a, b) -> { a + b }                    // inferred types
add_typed := (a: int, b: int) -> int { a + b } // explicit types
log := (msg) -> _ { print(msg) }               // void (-> _)
greet := () -> str { "hello" }                  // no args

// Mutable reference params with ::=
increment := (c ::= Counter) -> _ { c.value := c.value + 1 }
// compiles to: fn increment(c: &mut Counter) { ... }
// call sites auto-pass &mut: increment(c)
```

### Control Flow
```hom
// if/else (use `do` before `{`)
if (x > 0) do { print("positive") } else { print("non-positive") }

// match (expression, result can be assigned)
result := match dir {
  Direction.North => "up"
  Direction.South => "down"
  _ => "other"
}

// for loop
for i in range(10) do { print(i) }
for item in list do { process(item) }

// while loop (use ::= for mutable counter)
count ::= 10
while (count > 0) do { count ::= count - 1 }

// early return from loop
break => value
```

### Pipe Operator (chain functions)
```hom
result := @[1, 2, 3, 4, 5]
  | filter((x) -> { x > 2 })
  | map((x) -> { x * 10 })
  | reduce((a, b) -> { a + b })
// same as: reduce(map(filter(list, f), g), h)
```

### Structs (data only, no methods)
```hom
Point := struct { x: int, y: int }
p := Point { x: 10, y: 20 }
p.x := 15                      // field mutation

// For in-place mutation via functions, use ::= for both binding and params:
Counter := struct { value: int }
c ::= Counter { value: 0 }
increment := (c ::= Counter) -> _ { c.value := c.value + 1 }
increment(c)                   // c.value is now 1 (mutated in place)
```

### Enums
```hom
Direction := enum { North, South, East, West }
Shape := enum { Rect(str), Circle(int), Diamond }

match shape {
  Shape.Rect(label) => draw_rect(label)
  Shape.Circle(r) => draw_circle(r)
  Shape.Diamond => draw_diamond()
}
```

### Destructuring
```hom
a, b := b, a                   // swap (immutable rebind)
x, y := get_coords(node)       // unpack tuple return
_, second := get_pair()         // discard first

a, b ::= b, a                  // swap (mutable rebind)
```

### Imports
```hom
use std                         // standard library (range, len, filter, map, ...)
use heap                        // import heap.rs or heap.hom from same directory
use graph                       // import graph.rs or graph.hom
```

Import resolves `use foo` to: `foo.hom` → `foo.rs` → `foo/mod.hom` → `foo/mod.rs`
Search is relative to the calling file's directory.
If none found, passes through as Rust `use foo;`

Runtime libs (`heap`, `re`, `chars`, `str_ext`, `dict`) live in `Homun-Lang/runtime/`.
To use them in a project, either: (a) place the .hom test files in the same runtime/ dir,
or (b) copy/symlink the .rs files into the project's source directory.

### Error Handling (Result + ?)
```hom
parse := (src: str) -> Result {
  if (len(src) == 0) do { Err("empty") }
  else { Ok(build_ast(src)) }
}

// ? propagates errors (returns Err early)
node := parse_node(cursor)?
graph := build_graph(nodes)?
```

## Standard Library (`use std`)

### Collections
```hom
len(list)                       // length of list, dict, set, string
push(list, item)                // append to list
pop(list)                       // remove last
sorted(list)                    // sorted copy
reversed(list)                  // reversed copy
unique(list)                    // deduplicated
flatten(nested)                 // flatten nested lists
enumerate(list)                 // [(0, a), (1, b), ...]
zip(list_a, list_b)             // [(a1, b1), (a2, b2), ...]
index_of(list, item)            // position or -1
```

### Functional
```hom
list | filter((x) -> { x > 0 })       // keep matching
list | map((x) -> { x * 2 })          // transform each
list | reduce((a, b) -> { a + b })     // fold to single value
```

### Math
```hom
abs(x)                          // absolute value
min(a, b)                       // minimum
max(a, b)                       // maximum
clamp(x, lo, hi)                // clamp to range
```

### Strings
```hom
split(s, ",")                   // split by delimiter
join(list, ", ")                // join with separator
trim(s)                         // strip whitespace
replace(s, "old", "new")       // replace all
to_upper(s)                    // uppercase
to_lower(s)                    // lowercase
contains(s, "sub")             // substring check
find(s, "sub")                 // position or -1
substr(s, start, end)          // substring
char_at(s, idx)                // character at index (returns str)
lines(s)                       // split by newline
parse_int(s)                   // string → int

```

### Range
```hom
range(5)                        // 0, 1, 2, 3, 4
range(2, 7)                     // 2, 3, 4, 5, 6
range(0, 10, 2)                 // 0, 2, 4, 6, 8
```

## Runtime Libraries (import with `use`)

### `use heap` — Priority Queue
```hom
use heap
h := heap_new()
heap_push(h, 5, "item_a")      // priority (int), value (str)
heap_push(h, 1, "item_b")
result := heap_pop(h)          // Option<(int, str)> — lowest priority first
match result {
  Some((priority, item)) => { print("popped: ${item}") }
  none => { print("empty") }
}
n := heap_len(h)               // int
empty := heap_is_empty(h)      // bool
```
**API**:
- `heap_new() -> Heap`
- `heap_push(h, priority: int, item: str) -> _`
- `heap_pop(h) -> Option<(int, str)>`  — None if empty
- `heap_len(h) -> int`
- `heap_is_empty(h) -> bool`

### `use re` — Regex
```hom
use re
matched, text, end_pos := re_match("[a-z]+", src, pos)
if (matched) do { print("found: ${text}") }

ok := re_is_match("[0-9]+", "hello 42")   // true — matches anywhere in text
```
**API**:
- `re_match(pattern: str, text: str, pos: int) -> (bool, str, int)` — anchored at `pos`; returns (matched, matched_text, end_pos)
- `re_is_match(pattern: str, text: str) -> bool` — true if pattern matches anywhere

**Note**: Projects using `re` must add `regex = "1"` to Cargo.toml.

### `use str_ext` — String Extras
```hom
use str_ext
s := str_repeat(" ", 10)       // "          " — repeat str n times
s := str_pad_center("hi", 10)  // "    hi    " — center in width chars
```
**API**:
- `str_repeat(s: str, n: int) -> str` — n <= 0 returns ""
- `str_pad_center(s: str, width: int) -> str` — pads with spaces; odd padding goes right

### `use dict` — Dict Builder Helpers
```hom
use dict
// Build dict from list of (key, value) pairs
d := dict_from_pairs(@[("a", 1), ("b", 2)])

// Build dict from two parallel lists (replaces Python dict comprehensions)
d := dict_zip(keys, values)

// Clone a dict
d2 := dict_clone(d)
```
**API**:
- `dict_from_pairs(pairs: @[(K, V)]) -> @{K: V}` — last value wins on duplicate keys
- `dict_zip(keys: @[K], values: @[V]) -> @{K: V}` — stops at shorter list
- `dict_clone(d: @{K: V}) -> @{K: V}` — returns an independent copy

### `use chars` — Character Classification
```hom
use chars
is_alpha("a")                   // true (works with str or char)
is_alnum("3")                   // true
is_digit("7")                   // true
is_ws(" ")                      // true
```

## Style Guide

- Variables and functions: any naming style (no enforcement)
- Types: `PascalCase` (convention)
- Use `:=` for values that don't change, `::=` for values that will be reassigned
- Use `::=` in function params when the function needs to mutate the argument
- No methods — use free functions: `set(canvas, x, y, ch)` not `canvas.set(x, y, ch)`
- No classes — structs for data, functions for behavior
- Pipe `|` for chaining instead of method chains
- Last expression in `{}` is the return value (no `return` keyword)
- `break => value` for early return from loops

## Type Mapping (.hom → Rust)

| .hom | Rust | Notes |
|------|------|-------|
| `int` | `i32` | |
| `float` | `f32` | |
| `str` | `String` | |
| `char` | `char` | single quotes: `'a'`, `'\n'` |
| `bool` | `bool` | |
| `none` | `Option::None` | |
| `@[T]` | `Vec<T>` | |
| `@{K: V}` | `HashMap<K, V>` | |
| `@(T)` | `HashSet<T>` | |
| `-> _` | `()` | void |
| untyped param | `T` (generic) | Rust infers |

## Common Patterns for mermaid-ascii

### Struct as "object" with free functions
```hom
Canvas := struct { width: int, height: int, cells: @[] }

canvas_new := (w: int, h: int) -> Canvas {
  Canvas { width: w, height: h, cells: @[] }
}

canvas_set := (c: Canvas, x: int, y: int, ch: str) -> _ {
  c.cells[y][x] := ch
}

canvas_get := (c: Canvas, x: int, y: int) -> str {
  c.cells[y][x]
}
```

### Enum dispatch (replaces class inheritance)
```hom
NodeShape := enum { Rectangle, Rounded, Diamond, Circle }

draw_node := (shape: NodeShape, x: int, y: int) -> _ {
  match shape {
    NodeShape.Rectangle => draw_rect(x, y)
    NodeShape.Rounded => draw_rounded(x, y)
    NodeShape.Diamond => draw_diamond(x, y)
    NodeShape.Circle => draw_circle(x, y)
  }
}
```

### Recursive descent parser pattern
```hom
Cursor := struct { src: str, pos: int }

cursor_peek := (c: Cursor, s: str) -> bool {
  // check if src[pos..] starts with s
  substr(c.src, c.pos, c.pos + len(s)) == s
}

cursor_consume := (c: Cursor, s: str) -> bool {
  if (cursor_peek(c, s)) do {
    c.pos := c.pos + len(s)
    true
  } else { false }
}

cursor_eof := (c: Cursor) -> bool {
  c.pos >= len(c.src)
}
```
